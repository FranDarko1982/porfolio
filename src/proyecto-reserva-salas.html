Caso interno

Calendario único para reservas de salas de formación

Se centralizaron las reservas de varias sedes en una interfaz web única. El sistema controla permisos por rol, evita solapes y envía notificaciones automáticas de confirmación, cambios y cancelaciones.

Dolor

Excels y correos que nadie controla

Había colisiones constantes entre equipos al reservar salas. La coordinación dependía de correos y hojas de cálculo separadas por ciudad, con poca trazabilidad y sin vista consolidada.

Solución

Un calendario único y gobernado

  • Interfaz web con FullCalendar y búsquedas por centro, fecha y capacidad.
  • Roles (admin, responsable, formador) que determinan quién puede reservar o editar.
  • Notificaciones automáticas en cada evento y recordatorios 24h antes.
  • Panel de control con KPIs de ocupación y tiempos muertos por sala.

Integraciones

  • Google Apps Script + Sheets como base de datos ligera.
  • FullCalendar para visualización y gestión de eventos.
  • MailApp y plantillas HTML para avisos transaccionales.

Impacto

  • Cero solapes de salas desde el lanzamiento.
  • Reducción del 70% de correos de coordinación.
  • Visibilidad diaria de ocupación por sede y franja.
